1-2.CSSについて学ぼう！

CSSとはHTMLのデザインを変更することができる言語であり、色、大きさ、空白などの 要素を変えられます。HTMLの”どこの”、”何を”、”どのように”を明記することが必要です。 以下では、CSSの特性について復習しよう。

CSSの文法

CSSの文法は変更したい部分の宣言の後、次の行のはじめにはindentが必要であり、 何をの後に:、どのように の後に；が必要です。使い方を学んでいきましょう。

  • color

    • colorプロパティは16進数、または英語表記の色名を記述すればhtmlの要素の色を変えることができます。

  • font-size

    • font-sizeプロパティはフォントのサイズを変えることができます。通常、 フォントのサイズを変更する場合の単位はpx(pixel)です。

  • font-family

    • font-familyプロパティは"font-family:font name;" と設定することによって,フォントのタイプを明示することができます。

    フォント名が空白を含んでいる場合は、ダブルクォーツでフォント名を囲います。

  • background-color

    • background-colorプロパティは背景の色を変えることができます。#ddddddのように 同じ文字が続く場合は、#dddのように短く記述することができます。

  • width and height

    • widthプロパティとheightプロパティは要素の高さと幅を変えることができます。 これらのプロパティもfont-sizeプロパティと同様に単位としてpxが必要になります。

  • list-style

    • list要素にlist-styleプロパティを適用させることによって、様々な操作ができます。 例えば、"list-style:none;"とすることで箇条書きの黒丸を消すことができます。

  • float

    • 一般的にHTMLで書かれた要素は垂直に並びます。並行に要素を並べ替えたいときに floatプロパティは用いられます。"float:left;"とすれば、webページの 左側に並行に表示されます

  • padding

    • 要素ごとのスペースを適切な位置に調整したい場合は、paddingプロパティが便利です。 "padding:Npx"とすることで、四方端からNpx空間を作って要素を配置することができます。

    "padding-top","padding-right","padding-bottom","padding-left"と 空間を作る位置を一方だけに特定することもできます。

    四方位から指定したい場合は、上から時計回りで設定することができます。また、 上下と左右の２つを適用させることもできます。

  • border

    • 線の幅、タイプ、色を指定し、borderプロパティを用いるとボーダーを配置できます。 borderプロパティであれば四方に線が配置されますが、特定の場所にだけ配置したい場合は、 "border-top","border-right","border-bottom","border-left"で明示します。

  • margin

    • marginプロパティはpaddingプロパティと同様に空間を作るプロパティです。 指定する要素や書式も同じなので、２つの違いを説明します。

    例えば、borderプロパティで全方位に線を描いたとします。paddingプロパティは線の 内側の空間を定めるのに対して、marginプロパティは線の外側の空間を定めるプロパティです。

  • background-image

    • background-imageプロパティは背景に画像を挿入します。 "background-image:url(URL);"と明示します。

  • background-size

    • background-imageプロパティで配置した画像は、スペース全体を覆うまで繰り返し 配置されます。１つの画像だけで配置したいときは"background-size:cover;"とします。 (heightは定義する必要があります。)

  • opacity

    • opacityプロパティは透明な要素を作ることができます。引数は0.0から1.0まであり、 0.0が一番透明です。

  • letter-spacing

    • letter-spacingプロパティは文字の間隔を明示することができます。

  • display

    • displayプロパティは要素を"block","inline-block","inline"に変更できる。a tagは サイズや空間を変えることができないので、block elementにして変更するときに用いられる。

  • border-radius

    • border-radiusプロパティは要素の角を丸くします。数値が大きくなるほど、角は丸くなります。

  • text-align

    • text-alignプロパティはinline elementやinline-block elementを並び替えます。 引数は"left","center","right"です。

  • line-height

    • line-heightプロパティは１行が占める高さを明示することができます。高さは変更されても 要素は中心に配置されます。

  • font-weight

    • font-weightプロパティはテキストの厚さを変えることができます。引数はnormalや boldです。

    h1~h6はデフォルトでboldに設定されています。

  • box-shadow

    • box-shadowプロパティは要素に影を加える場合に使われます。水平方向の長さ、 垂直方向の長さ、色の３つの引数を指定することによって使うことができます。

  • z-index

    • z-indexプロパティは要素の積み重ねに順番を決めることができます。数値の大きい方が上面に 表示されます。z-indexはpositionプロパティと合わせて使うことに注意してください。

特定の要素にCSSを適用させる方法

class属性を用いてラベル化された要素だけを変更することが可能になります。HTNLのタグに classを加え、CSSでクラス名を明示する。このとき、クラス名のはじめに. が必要になるので忘れないように。

また、複数のHTMLの要素に同じクラス名をつけると、CSSでの変更が同じクラス名 の要素全てに適用されます。異なるtag同士に同じプロパティを適用させたい場合は、 複数のtagをコンマ(,)で区切ります。

The Box Model

線で要素を囲うときは、borderとpaddingとmarginの３つのプロパティを用いて作ります。

Submit Botton

input要素の中にtype属性を加えることによって送信ボタンを作ることができます。 type属性をsubmitにすることでinput要素が送信ボタンになります。表示されるテキストは デフォルトでSubmitになっています。もし、テキストの内容を変えたい場合は value属性で変更してください。

margin:0 autoとtext-align:centerの違い

要素を中心に配置できるプロパティとしてmargin:0 autoとtext-align:centerを学びました。 marginプロパティはcontainersのようなblock elementに用います。一方、text-alignプロパティは ボタンやテキストなどのinlineやinline-block elementに用います。

2.HTML&CSSでウェブサイトを作る流れについて

2-1.HTML&CSSの制作手順

コメントについて

多くのプログラミング言語ではコメントを残したいときの記法があります。HTMLの場合 テキストを＜！ーー（テキスト）ーー＞で囲うとコメントを 残すことが可能です。

また、CSSの場合は/*（テキスト）*/で囲うとコメントを残すことが可能です。

実際のHTMLテキストは以下のようなルールがあります。確認していきましょう。

  1. ＜head＞、＜body＞要素が＜HTML＞要素の中に必要とされています。＜head＞には テキストの情報を記載します。＜body＞はディスプレイで表示される内容を記載します。
  2. 全てのHTMLテキストは＜！DOCTYPE html＞を宣言してから始めなければいけません。 （HTMLのバージョンによって異なりますが、最新のHTMLなら問題ありません。）
  3. ＜head＞要素に記載されるHTMLの設定はブラウザ上には表示されません。ここには文字コード 、ウェブのタイトル、CSSファイルのリンクの３つを明示する必要があります。 （覚える必要はありませんが、意味を理解する必要があります。）
    • 文字コードとして、charset属性に＜meta charset="utf-8"＞を明示することに よって文字化けを防ぎます。
    • title tag内にあるテキストはブラウザのタブで表示されます。
    • HTMLでhref属性[href="（CSSの名称).css"]の記載と＜link rel="stylesheet"＞を定義することによって HTMLとCSSを関連させてウェブを作ることが可能になります。
  4. レイアウトはウェブを作る上で、最も重要な部分の１つです。レイアウトは主に３つの 部分で構成されています。
    • header

      • headerにはlogoとlistをdivでグループ分けして表示します。その後、logoとlistを floatプロパティやpaddingプロパティで調整していきます。

    • main

      • div tagで"copy-container","contents","contact-form"をクラス分け していきます。それぞれの部分ごとにclassで分類し、CSSでコーディングしていきます。

    • footer

      • hooterにも同様にlogoとlistを配置していきます。footerは"float:left;"で 右側に配置します。またheaderとfooterでlogoとlistを配置する場所が異なります。 この場合、CSSでclass名の後に、スペースを入れtagを記載することで、特定の要素だけ 変更を加えることができます。

3.HTML&CSSの応用編

3-1.HTML＆CSSの応用

  • header tagとfooter tag

    • 前のレッスンではdiv tagを用いてclassでheaderとfooterを作りました。 しかし、一般的にはheader tagとfooter tagがよく使われます。

  • 要素を中心に配置する

    • marginのleftとrightをautoとすることによって要素を中心に配置することができます。 このとき、widthは明示していないといけません。

  • anchor tagを利用する

    • anchor tagはサイズや空間を明示することができません。この問題を解決する方法として a tagで作った要素はinline elementであり、これをblock elementにすることで サイズやスペースを明示できます。a tagはデフォルトでinline elementsなのでdisplay プロパティで変更することができます。

  • 複数のクラスで同じプロパティを適用する方法

    • CSSのプロパティがわずかに異なるいくつかの要素があるとき,共通のCSSと個々のCSS を異なるクラスとして抜き出すことができます。クラス名をスペースで分けることで 複数のクラスを明示できます。

  • アイコンフォントを使う方法

    • head tag内でアイコンフォントをダウンロードすれば、アイコンフォントが使えるようになります。 使用するときはspan tag内でfa classとfa-"iconname" classを明示します。

  • rgbとrgba

    • rgbは３つの引数の組み合わせで色を明示する方法です。（例えば、rgb(73,201,192) と#49C9C0は同色）プログラミングでは１６進数で色を表します。

    rgbaは色の透明度を明示したい場合に使われます。４つ目の引数は透明度で0.0から1.0までで、 0.0が一番透明です。

  • login linkの作り方

    • anchor tagを用いてログインボタンは作れます。このとき、herf属性は"ログイン先のURL"としてください。

    また、anchor tagはinline elementなのでテキスト部分しかクリックすることができません。なので "display:blockb"として、block element全体がクリックできるように修正が必要です。

  • サイズを％で明示する

    • pxの代わりに％を使うと、parent elementの幅のサイズから計算してサイズを決めることができます。 高さに関しても適用できます。

  • 要素を重ねる

    • 通常のHTMLでは、要素を重ねることはできません。CSSで"position:absolute"を使えば、 要素の上に要素を重ねることができます。要素の左上をtopプロパティとleftプロパティで 明示して位置を決めることができます。

    CSSで、parent element classに"position:relative"を明示することで、 "position:absolute"で決めたtopプロパティとleftプロパティをparent elementの 左上を基準に動かすことができます。

  • 要素を固定する

    • 画面を下へスクロールしたとき、画面の上部にヘッダーを固定したい場合は、どうすればいいか を学んでいきます。"position:fixed"を使うとスクリーンの特定の位置に要素を固定することができます。 固定する位置はtopプロパティとleftプロパティで指定できます。
